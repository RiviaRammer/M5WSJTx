# Installation

- **Board:** M5Stack (3.2.6, mic is disable on 3.3.7)  
- **Libraries:** M5Cardputer (and all required dependencies)

# Setting

- **Partition Scheme:** 
- **Flash Size:** 8MB(64Mb)  
- **PSRAM:** QSPI PSRAM  

# Mode

- FT8  

# Usage

- **ESC**: Abort the current task  
- **Fn + M**: Switch modulation/demodulation mode  
- **/**: Change Home/Message/Waterfall
